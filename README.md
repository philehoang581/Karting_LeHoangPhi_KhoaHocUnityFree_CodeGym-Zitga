Game đua xe dựa trên Asset Karting để xây dựng thêm tính năng:
1. Đăng nhập người chơi.
2. UI main gồm cách button: Shop, Map, Car, Play...

4. Và những tính năng khác xin vui lòng xem File ReadMe
